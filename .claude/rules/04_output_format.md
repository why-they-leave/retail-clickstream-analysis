# Rule 04 — 출력 형식

## 코드 블록
- 언어 태그 필수: ```python, ```bash, ```sql
- 실행 가능한 완전한 코드만 제공 (부분 스니펫 지양)
- 긴 코드는 섹션 주석으로 구분

## 분석 결과 요약 형식
```
## 분석 결과

**핵심 발견:**
- 발견 1
- 발견 2
- 발견 3

**수치 요약:**
| 지표 | 값 |
|------|-----|
| ... | ... |

**해석:** (2-3문장)

**권장 다음 단계:** (선택)
```

## 숫자 표기
- 소수점 이하 3자리까지 표시 (통계 수치)
- 퍼센트는 소수점 1자리
- 1000 이상 숫자는 쉼표 구분: `1,234,567`

## 시각화 설명
- 그래프 제목, x/y 축 레이블, 범례 필수
- 색상은 색약 친화 팔레트 우선 (`colorblind` safe)
- 한국어 폰트 설정 코드 포함

## 에러 응답
- 코드 에러 발생 시 원인을 한 줄로 먼저 설명
- 수정된 코드 전체를 다시 제공 (diff만 제공 지양)
